# Boccia MCU

This project contains the files for the firmware portion of the Boccia project.
The folders are divided into the following:
- Boccia-controller. The Boccia controller folder is a [PlatformIO](https://platformio.org/platformio-ide) project that includes all the necessary files for the firwmare
- [Examples](./Examples/). Has multiple subfolders with ArduinoIDE projects to test individual portions of the firmware
